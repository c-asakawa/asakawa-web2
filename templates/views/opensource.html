Fish Tank Random Number Generator
Goto The Repo This project uses a Raspberry Pi 3 with a camera to create a live stream of my fish tank. This live stream is then used to generate a random number.
Fish Tank Setup
We used a raspberry pi camera to stream a video of the fish. We then process this video to give us an x and y position of each fish for each frame of the video.
Translating Fish Movement
Fish movements are tracked by their position on an x/y plane, this change of position is used to generate a binary number as shown on the the picture to the left.
Random Distribution
This is a scatter plot graph that shows the distribution of about 2000 randomly generated x and y values between 0 to 128, generated from the fish.
